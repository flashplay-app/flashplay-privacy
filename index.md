**Última atualização:** 11/01/2026

## 2. Dados que coletamos e tratamos
O FlashPlay **não exige login** e **não coleta dados de conta** (como e-mail, senha ou nome de usuário).

Para fornecer o serviço, tratamos apenas:

### 2.1. Vídeos
- O FlashPlay armazena vídeos gerados pelo sistema para permitir que você visualize e compartilhe lances.
- Os vídeos são armazenados na nuvem (Amazon Web Services — **Amazon S3**).

### 2.2. Metadados técnicos mínimos
- Informações mínimas necessárias para organização e funcionamento, como **data/hora de criação** e **identificadores internos** do arquivo.

### 2.3. O que NÃO coletamos
- **Não coletamos localização**.
- **Não utilizamos ferramentas de analytics** (por exemplo, Firebase Analytics).
- **Não coletamos dados de login/senha**, pois não há conta no app.

## 3. Finalidade do tratamento
Usamos os dados exclusivamente para:
- armazenar e disponibilizar os vídeos no aplicativo;
- manter o funcionamento e a entrega dos vídeos.

## 4. Armazenamento e retenção
- Os vídeos ficam armazenados por **até 10 dias** e depois **são removidos automaticamente**.

## 5. Compartilhamento com terceiros
Compartilhamos dados **apenas** com fornecedores necessários para operar o serviço, como:
- **Amazon Web Services (AWS) — Amazon S3** (armazenamento dos vídeos)

Não vendemos dados.

## 6. Segurança
Adotamos medidas técnicas razoáveis para proteger os dados, incluindo controle de acesso ao armazenamento e boas práticas de segurança.
